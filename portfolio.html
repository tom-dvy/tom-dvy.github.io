Lego FC Barcelone

Lors d'un workshop à l'IIM, nous avons eu pour mission de modéliser un personnage LEGO en une semaine sur le logiciel Blender.

En tant que supporter, j'ai choisi de créer une version LEGO aux couleurs du FC Barcelone. Ce projet m'a permis de développer mes compétences en modélisation 3D, texturing et rendu.

Mon rôle : Modélisateur 3D, Texturing

Outils utilisés : Blender


Hyrome Fighter

Projet de fin de première année de BTS, "Hyrome Fighter" est un jeu de combat 2D développé en C# avec le framework Monogame.

J'étais en charge de la programmation des mécaniques de combat, de l'intelligence artificielle de l'adversaire et de l'intégration des assets graphiques.

Mon rôle : Développeur Gameplay

Outils utilisés : Scratch


Affiche "Nantes Digital Week"

Création d'une proposition d'affiche pour l'événement "Nantes Digital Week" dans le cadre d'un cours de communication visuelle. Le travail a été réalisé sur Adobe Photoshop et Illustrator.

L'objectif était de représenter l'innovation numérique en y intégrant un symbole fort de la ville de Nantes, ici le stade de la Beaujoire.

Mon rôle : Graphiste

Outils utilisés : Adobe Photoshop


Dashboard Énergétique

Développement d'une interface web pour la visualisation de données de consommation énergétique. Ce projet a été réalisé en utilisant HTML, CSS et la librairie JavaScript Chart.js.

Les données sont chargées dynamiquement et affichées sous forme de graphiques interactifs pour faciliter leur analyse.

Mon rôle : Développeur Front-End

Outils utilisés : Eclipse


IHM Grainothèque

Conception et prototypage d'une Interface Homme-Machine (IHM) pour une grainothèque connectée. Le but était de créer une interface simple et intuitive pour gérer les stocks de graines.

Les maquettes ont été créées sur Figma, en se concentrant sur l'expérience utilisateur (UX) et le design d'interface (UI).

Mon rôle : Développeur Front-End/Back-End

Outils utilisés : VS Code


Pokédex

Conception d'une application Pokédex fonctionnelle. Le but était de créer une interface simple et intuitive pour consulter les pokémons.

Je me suis aidé de l'api PokeAPI pour récupérer les données. L'utilisateur a accès aux différentes formes, aux évolutions, aux stats, aux types, ect de chaque pokémon, via une fiche technique.

Mon rôle : Développeur Front-End/Back-End

Outils utilisés : PokeAPI
